# 🎓 Student Management System (Flask)

A simple and clean **Flask-based web application** for managing student records with authentication system.

---

## 🚀 Features

- 🔐 User Signup & Login (Authentication)
- 👨‍🎓 Add Students
- 📋 View Students List
- ✏️ Edit Student Details
- ❌ Delete Students (Soft Delete)
- 📄 View Individual Student Details
- 🧾 Flash Messages for user feedback
- 💅 Clean UI using Bootstrap

---

## 🛠️ Tech Stack

- Python 🐍
- Flask 🌐
- SQLite 🗄️
- HTML + CSS + Bootstrap 🎨
